# Legion Services LLC Website

Astro website for Legion Services LLC with Electrical, Aerial Operations, and Real Estate division pages.

## Publish through GitHub and Cloudflare Pages

1. Upload the contents of this folder to the root of the connected GitHub repository.
2. Do not upload this folder inside another nested folder.
3. Remove any old `postcss.config.cjs` or Tailwind configuration files from the repository; this site does not use Tailwind.
4. Commit directly to the production branch, normally `main`.
5. Cloudflare Pages should build automatically with:
   - Build command: `npm run build`
   - Output directory: `dist`
   - Framework preset: Astro
